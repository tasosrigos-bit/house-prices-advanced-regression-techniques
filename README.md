This project is a part of the [House Prices: Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques) competition on Kaggle. The goal of this competition is to predict the sale price of a house based on its features. The dataset contains various features of the houses, such as the number of bedrooms, the size of the house, the year it was built, and many more.

In this project, we will explore different regression techniques to predict the sale price of the houses. We will start with a simple Linear Regression model and then move on to more complex models such as Ridge Regression, Lasso Regression, and Random Forest Regression. We will also perform feature engineering and hyperparameter tuning to improve the performance of our models.
